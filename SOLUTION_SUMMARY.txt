📝 EXAMPLE USAGE:

Browser (Easiest):
   1. Open: http://localhost:8000/docs
   2. Click "/generate"
   3. Click "Try it out"
   4. Enter: {"prompt": "Which is the largest country?", "max_new_tokens": 50}
   5. Click "Execute"
   6. See response!

Python:
   import requests
   r = requests.post('http://localhost:8000/generate', json={
       'prompt': 'Which is the largest country?',
       'max_new_tokens': 50
   })
   print(r.json()['generated_text'])

JavaScript/Frontend:
   const r = await fetch('http://localhost:8000/generate', {
     method: 'POST',
     headers: { 'Content-Type': 'application/json' },
     body: JSON.stringify({
       prompt: 'Which is the largest country?',
       max_new_tokens: 50
     })
   });
   const data = await r.json();
   console.log(data.generated_text);

❓ FREQUENTLY ASKED QUESTIONS:

Q: Why did the original backend fail?
A: The Hugging Face Inference API returned 404 errors for the models.
   We switched to local transformers which work offline and reliably.

Q: Do I need a GPU?
A: No! The solution uses CPU mode. It's slower but works everywhere.

Q: Why is the first request slow?
A: The AI model (GPT2, 500MB) is downloaded and loaded on first use.
   Subsequent requests are much faster (2-5 seconds).

Q: Can I use it in production?
A: Yes! main_local.py is production-ready. Scale with:
   - Load balancer (nginx)
   - Multiple instances
   - GPU acceleration (change device='cuda')

Q: How do I change the model?
A: Edit .env and change HF_MODEL value.
   Examples: gpt2, distilgpt2, other HF models

Q: Can I add more features?
A: Yes! main_local.py is a FastAPI app. Easy to extend.

🆘 HELP & TROUBLESHOOTING:

Connection Refused?
   → Make sure python main_local.py is running
   
Port 8000 in use?
   → Change API_PORT in .env to 8001, 8002, etc.

Slow first request?
   → Normal! Model is loading. Just wait 1-2 minutes.
   → Subsequent requests are 2-5 seconds.

Out of memory?
   → Use distilgpt2 model instead of gpt2
   → Reduce max_new_tokens parameter

More help?
   → See: README.md, EXAMPLES.md, QUICK_START.md
