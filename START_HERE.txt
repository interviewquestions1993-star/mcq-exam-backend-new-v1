💻 USAGE EXAMPLES:
================================================================================

Python:
  import requests
  r = requests.post('http://localhost:8000/generate', json={
    'prompt': 'What is AI?',
    'max_new_tokens': 100
  })
  print(r.json()['generated_text'])

JavaScript:
  const r = await fetch('http://localhost:8000/generate', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      prompt: 'What is AI?',
      max_new_tokens: 100
    })
  });
  const data = await r.json();
  console.log(data.generated_text);

PowerShell:
  $body = @{ prompt='What is AI?' } | ConvertTo-Json
  Invoke-WebRequest -Uri 'http://localhost:8000/generate' `
    -Method POST -Body $body -ContentType 'application/json'

🐛 TROUBLESHOOTING:
================================================================================

❌ "Connection refused"
   → Backend not running. Run: python main.py

❌ "Model is loading..." (first request slow)
   → Normal! First request takes 30-60s. Just wait.

❌ "HF_TOKEN not found"
   → Check .env file has your token

❌ "Rate limited" (429 error)
   → Backend retries automatically. Just wait.

❌ Can't open http://localhost:8000/docs
   → Port 8000 might be in use. Change API_PORT in .env

See README.md for more troubleshooting.
